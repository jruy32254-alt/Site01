Conteudos



HTML5 é a quinta e mais recente versão da linguagem de marcação de hipertexto (HTML), usada para estruturar o conteúdo de páginas web. Ele introduz novos elementos e funcionalidades que facilitam o desenvolvimento de sites modernos, com foco em interatividade, multimídia e compatibilidade com dispositivos móveis.

CSS, ou Folhas de Estilo em Cascata, é uma linguagem usada para descrever a apresentação de documentos HTML ou XML. Em termos simples, o CSS define como os elementos de uma página web devem ser exibidos, incluindo cores, fontes, layout e outros aspectos visuais. O CSS trabalha em conjunto com o HTML, que define a estrutura do conteúdo, enquanto o CSS adiciona estilo e apresentação

O que é a propriedade Display em Css


A propriedade display permite que você defina a maneira como determinado elemento HTML deve ser renderizado.

display caracteristicas
Block Para blocos principais de conteudo, seçoes e elementos q precisam de sua prorpia linha e controle atotal de tamanho. pense em containers e estruturas verticais
Inline Ideal para pequenaos elementos dentro de um fluxo de texto, como links, negritos, italicos , onde voce nao precisasr controlar largura e altura.
flex define um elemento como um container flexível, permitindo o uso do layout flexbox para organizar seus filhos (elementos filhos). O flexbox é uma ferramenta poderosa para criar layouts responsivos e flexíveis, tanto horizontal quanto verticalmente, com controle sobre o alinhamento, espaçamento e redimensionamento dos itens.
